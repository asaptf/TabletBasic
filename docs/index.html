TabletBasic Support

TabletBasic is a retro BASIC learning environment for iPhone and iPad. This page provides support information for questions, feedback, and bug reports.

Contact

For support requests, questions, or feedback, use one of the options below:

  • GitHub Issues: Report a bug or ask a question
  • Project repository: github.com/asaptf/TabletBasic

Getting Started

  • On launch, press Enter to open the learning guide or Esc to dismiss the welcome dialog.
  • Use File > Open Sample Program to load built-in BASIC examples.
  • Use Run > Start to execute the current program.
  • Try immediate mode for quick experiments with short BASIC statements.

Frequently Asked Questions

Does TabletBasic require an account or internet connection?
No. The app runs locally on your device and does not require login for normal use.

Is TabletBasic affiliated with Microsoft or Apple?
No. TabletBasic is an independent educational project inspired by classic BASIC environments.

Where are my programs stored?
Programs stay on your device unless you save or export them using the system file picker or iCloud Drive.
